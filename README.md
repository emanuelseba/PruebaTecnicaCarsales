# PruebaTecnicaCarsales

### Backend

Entrar a la carpeta backend

```bash
cd backend/
```

Iniciar en local utilizar este comando

```bash
dotnet run
```

### Frontend

Entrar a la carpeta backend

```bash
cd frontend/
```

Instalar librerias

```bash
npm install
```

Arrancar en local utilizando este comando

```bash
ng serve
```
